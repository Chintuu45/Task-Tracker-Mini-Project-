# Task Tracker

A simple task management app built with HTML, CSS, and JavaScript. Add, edit, and delete tasks easily.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Responsive design

## Usage

1. Enter a task and click "Add Task"
2. Click "Edit" to modify a task
3. Click "Delete" to remove a task
